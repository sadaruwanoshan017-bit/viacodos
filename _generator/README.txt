VIA CODOS SITE GENERATOR
========================
All page copy lives in data.py; page templates live in build.py.
To edit content and regenerate every page:

  1. Edit data.py (services, industries, case studies, insights, team, jobs)
  2. Run:  python build.py

All HTML files in the project root are regenerated in place.
css/, js/ and assets/ are never touched by the generator.
